# computer_operating_status

## 背景

研究室のローカルネットワーク内でクラスターごとの稼働率やディスク使用率を確認できるWebアプリケーションを作った. <br/>

## 使用技術(ライブラリ等）

- d3.js
- jQuery
- kyotocabinet
